## INSTALLATION :

1. clone the git repository

```
git clone https://github.com/lokesharma95/Book-api.git
```

2. Run composer command to install project dependencies

```
composer install
```

3. migrate database

```
php artisan migrate
```

4. Run php server using

```
php artisan serve
```

5. Run test cases using

```
.\vendor\bin\phpunit.bat
```
